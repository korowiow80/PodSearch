Technik

  • Crawler Python + urllib.urlretrieve()
  • Suche Java + Lucene
  • Suchserver Solr + Tomcat / Jetty / Netty
  • Webserver lighttpd / nginx / G-WAN
  • Client HTML, CSS, JS + JQuery

Herausforderungen

  1. Heterogene Daten Audio-Dateien über Feeds indexieren
  2. Heterogene Datenquellen Verzeichnisse scrapen
  3. Interaktivität Suchergebnisse mittels AJAX holen
  4. Integrität Neue Feeds auf Datenverlust prüfen
  5. Sicherheit Benutzereingaben auf enthaltene Attacken prüfen
  6. Skalierbarkeit eigene Server für Suche und statischen Inhalt
  7. Offenheit Benutzereingaben von Feeds über Webinterface
  8. Adaption Layout mittels Media Queries anpassen

Außerhalb des Projektfokus

liegt Folgendes:
  • Podcast-Player Podcast-Sites haben meist schon Player
  • Podcast-Seiten Podcasts haben laut RSS eine Site zu haben
  • Benutzerprofil ist für Funktionalität nicht nötig
  • Soziale Netze integrieren meist schon die Podcast-Sites
  • Ontologie Tags lassen sich nicht streng hierarchisch ordnen
